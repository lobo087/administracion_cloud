# Administracion Cloud UTPL

Repositorio para organizar el material revisado en la materia de Administracion Cloud.

Semana a semana se iran subiendo carpetas con lo revisado en cada semana, incluyendo ejercicios, apuntes y demas recursos trabajados.

## Estructura

- `SEMANA_1/`: contenido correspondiente a la primera semana.
- `SEMANA_2/`: ejercicios con Docker Compose, MySQL, persistencia y contratos inteligentes con Solidity.
